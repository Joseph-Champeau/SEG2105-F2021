Test Cases
Some of many tests cases addressed for our submission are the following;

All the buttons are present and text written on them is readable.
The arithmetic operations are working fine- +, -, /, *.
The calculator gives the correct result in case of operations containing decimal numbers.
Hitting ‘C’ cancels the last digit or operation added.
Pressing two operators one after the other, the latest one will override the previous operator.
An integer output will display as an integer with no decimals.
Simply clicking "=" will not make the program crash.
